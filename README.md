   εξής "Server//test/file1" το οποίο θα προκαλέσει πρόβλημα.
3) Η δημιουργία του καταλόγου προς αντιγραφή <directory> γίνεται μέσα σε έναν άλλο φάκελο "output" τον οποίο κάνω define
   στο 'remoteClient.cpp'.
